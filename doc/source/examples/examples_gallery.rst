Tips for Adding Media
---------------------

- Place images/videos in ``source/_static/examples/`` for proper Sphinx integration.
- Use relative paths when referencing media files.
- Prefer ``.png`` for images and ``.mp4`` for videos.
- Optimize file size to keep documentation lightweight.
